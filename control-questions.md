Ответы на контрольные вопросы
Операционная система macOS и архитектура
Какая текущая версия macOS?
Актуальная стабильная версия на текущий момент (весна 2026) — macOS 16 (название зависит от релиза Apple, на 2025 год актуальной была macOS 15 Sequoia).

На основе какой UNIX-подобной системы построена операционная система macOS?
macOS построена на основе FreeBSD и Mach (в составе открытой основы Darwin).

К какому типу относится ядро macOS? В чем отличия между XNU и Mach?

Ядро macOS (XNU) является гибридным.

Mach — это микроядро, отвечающее за базовые вещи: управление памятью и передачу сообщений.

XNU (X is Not Unix) — это комбинация Mach и компонентов FreeBSD. В отличие от "чистого" Mach, XNU включает в себя высокоуровневые функции (драйверы, сетевой стек) прямо в пространстве ядра для скорости.

Какой тип архитектуры у процессоров линейки M1 и выше?
Процессоры Apple Silicon используют архитектуру ARM (AArch64).

Что такое виртуализация? Приведите примеры гипервизоров.
Виртуализация — это технология создания программного представления (виртуального) чего-либо (процессоров, ОС, сетей).

Примеры: VMware, VirtualBox, Parallels Desktop, QEMU.

Работа в терминале
Что такое «командная оболочка»?
Это программа-интерпретатор, которая принимает команды от пользователя в текстовом виде, передает их операционной системе и выводит результат.

Как называется командный интерпретатор и какая его версия в используемой системе?
В современных macOS стандартом является zsh (Z shell). Версию можно узнать командой zsh --version.

Для каких задач предназначено приложение Terminal в macOS?
Для прямого взаимодействия с ОС через текстовый интерфейс: управления файлами, настройки системы, компиляции кода и работы с Git.

Полный путь к домашней папке и команда для его определения.

Путь: /Users/имя_пользователя (например, /Users/mikhail).

Команда: pwd (print working directory) или echo $HOME.

Команда для вывода содержимого каталога системных конфигураций?

Bash
ls /etc
Установка Midnight Commander (mc) тремя способами через консоль:

brew install mc (через Homebrew).

sudo port install mc (через MacPorts).

Сборка из исходников (последовательность configure, make, make install).

Как вывести справочную информацию для команды?
Команда man (например, man ls).

Для каких целей используется команда open?
Для открытия файлов, папок или приложений так же, как если бы вы дважды кликнули по ним в Finder (например, open . откроет текущую папку).

Примеры команд группировки и перенаправления в zsh:

> — перенаправление вывода в файл (с перезаписью).

>> — добавление вывода в конец файла.

| (pipe) — передача вывода одной команды на вход другой.

{ } — группировка команд.

Команда изменения оболочки с zsh на bash:

Bash
chsh -s /bin/bash
Работа с Git
В каких случаях выполняется инициализация репозитория? Какой командой?
Когда нужно начать отслеживать изменения в новом или существующем проекте. Команда: git init.

Что такое коммит?
Это "снимок" (snapshot) состояния файлов проекта в определенный момент времени с уникальным идентификатором.

Почему важно создать коммит перед созданием новой ветки?
Чтобы зафиксировать текущие изменения. Это гарантирует, что новая ветка начнется с "чистого" и понятного состояния кода, и вы не потеряете незавершенную работу.

В каких состояниях может находиться файл в репозитории?

Untracked (неотслеживаемый).

Staged (подготовленный к коммиту после git add).

Unmodified (неизмененный после коммита).

Modified (измененный).
Изменение происходит через команды git add (в индекс) и git commit (в историю).

Что такое ветка?
Это подвижный указатель на один из коммитов, позволяющий вести разработку параллельно основной линии.

Обязательное условие для формирования ветки master?
В репозитории должен быть сделан хотя бы один (первый) коммит.

Что такое HEAD?
Это указатель на текущую ветку или коммит, с которым вы сейчас работаете.

Способы создания веток:

git branch имя_ветки — создать.

git checkout -b имя_ветки — создать и сразу перейти.

Как узнать текущую ветку?
git branch (текущая отмечена звездочкой) или git status.

Как переключаться между ветками?
git checkout имя_ветки или git switch имя_ветки.

Команда для слития веток:
git merge имя_ветки (находясь в той ветке, в которую нужно влить изменения).

Как подключить внешний репозиторий?
Команда: git remote add origin URL_репозитория.

Команда для отмены последнего коммита?
git reset --soft HEAD~1 (удалит коммит, но оставит изменения в файлах).

Для чего предназначена команда rebase?
Для переноса базы текущей ветки на другой коммит (линеаризация истории коммитов).

В каких случаях требуется создание форка? Способы создания.
Форк нужен, если вы хотите внести правки в чужой проект, к которому у вас нет прав доступа на запись.

Способы: кнопка Fork в интерфейсе GitHub или через консольную утилиту GitHub CLI: gh repo fork URL.
